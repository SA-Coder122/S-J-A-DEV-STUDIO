## 🎯 CUSTOMIZATION GUIDE

### Easy Customizations

**1. Change Colors**

- Edit `css/main.css` lines 8-23
- Update CSS variables in `:root`
- All colors update automatically

**2. Change WhatsApp Number**

- Edit `js/main.js` line 4
- Update phone number (without +)
- All booking buttons update

**3. Change Hotel Info**

- Find text in HTML files
- Replace with your content
- Images optional

- Save images to `assets/` folder
- Update image paths in `gallery.html`
- Or replace gradient backgrounds

**5. Update Room Details**

- Edit `rooms.html`
- Add/remove room types
- Update features list
